Circulation and Reference Services.

Library circulation includes checking out library materials to library users,renewing the borrowed items,reserving checked out items for the patron,checking in materials returned,checking materials for damage at the time of return, if found damaged then it is given to the staff responsible for repair.Re-shelving the library materials by call number given.

Online Reservation Of Books.

Customers can reserve books using normal online reservations process or by calling customer contact centre.To reserve you need to be a library member.Can be reserved online via library catalogue.You can also select the desired library from which you would like to pick the item.Exceptions are made to reference books and special collections.
